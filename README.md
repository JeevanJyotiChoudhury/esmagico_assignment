# esmagico_assignment

### Installation

1. Clone the repository:

   bash
   git clone https://github.com/JeevanJyotiChoudhury/esmagico_assignment.git
   

2. Change to the project directory:

   bash
   cd esmagico_assignment
   

3. Install dependencies:

   bash
   npm install

### Tech-stack Used

1. ReactJS
2. Chakra UI
